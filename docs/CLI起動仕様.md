# CLI起動仕様

## 目的

実行ファイル起動時に、利用者が状態を把握しやすく、設定不備を自己解決しやすいCLI表示を提供します。

## 対象

- 実行ファイル起動時のCLI表示
- 起動前後の案内メッセージ
- 引数による基本オプション

## CLIオプション

- `--help`
- `-h`
- `--version`

### `--help` の表示内容

- 使い方
- 利用可能なオプション一覧
- 起動前に必要な準備（`.env` 配置、`BOT_TOKEN` 設定）

### `--version` の表示内容

- アプリケーション名
- バージョン文字列

## 起動時の表示仕様

通常起動時は、起動処理の進行状況を段階的に表示します。

- 起動情報（アプリ名、バージョン）
- 実行ディレクトリ
- ログファイル保存先
- 起動ステップの進行状況（例: `1/4`, `2/4` の形式）
- 接続開始の案内
- 終了操作の案内（`Ctrl+C`）

## エラー表示仕様

エラー発生時は、原因だけでなく対処方法を含めて表示します。

### `.env` が見つからない場合

- `.env` の不足を明示
- 配置場所（実行ファイルと同じフォルダ）を案内
- 設定例（`BOT_TOKEN=...`）を表示

### `BOT_TOKEN` が未設定の場合

- 必須設定不足であることを明示
- `.env` に追記する設定例を表示

### その他の起動エラー

- エラー種別を明示（起動失敗 / 実行中エラー）
- 利用者に見える範囲で必要な文脈を表示（例: 対象ファイルパス）

## 終了コード

- `0`: 正常終了
- `1`: 起動失敗または実行中エラー
- `2`: 引数エラー

## 非対象

- Bot本体の動作仕様
- Discord上の機能仕様
- 内部実装詳細（関数分割、ログ実装方法、テスト構成など）
