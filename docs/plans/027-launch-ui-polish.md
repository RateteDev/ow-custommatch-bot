# 027-launch-ui-polish

## Context

起動 UI は 022 で大きく改善されましたが、ユーザー確認では
バナー内レイアウト、保存先表示項目、接続成功後の案内文に追加改善案が出ています。

細かな文言と視線誘導を見直し、起動後すぐに次の操作が分かる UI へ仕上げます。

## 変更ファイル

| ファイル | 対応内容 |
|---------|---------|
| `cmd/ow-custommatch-bot/main.go` | バナー、保存先表示、接続成功メッセージを改善 |
| `cmd/ow-custommatch-bot/main_test.go` | 表示文言テストを更新 |
| `docs/CLI表示仕様.md` | 最終表示仕様へ更新 |

## 実施順序

1. バナー内情報の配置を見直す
2. 保存先表示の項目数を整理する
3. 接続成功後の案内文を追加する
4. テストと仕様書を更新する

## 要件

### 要件1: バナーの情報集約

- ツール名、バージョン、使い方 URL を囲み枠の中で見せる

### 要件2: パス表示の整理

- `データ保存先` を残すか削るかを決め、表示項目を最適化する

### 要件3: 接続成功メッセージ

- Discord ログイン成功後に `/match` 開始案内まで表示する

## 検証方法

- `go test ./cmd/ow-custommatch-bot/...`
- 起動時の表示と接続成功後のメッセージを手動確認
