# ROADMAP

ow-custommatch-bot の改善・機能追加ロードマップ。
各プランの詳細は `docs/plans/` を参照。

## 進め方

各アイテムは対応する plan ファイルをもとに `/subagent` で実装依頼する。
優先度の高いものから順に着手し、1 プランずつ完了させてから次へ進む。

---
