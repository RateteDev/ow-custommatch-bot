ow-custommatch-bot Third-Party Notices
======================================

This file lists the primary third-party Go modules used by the Windows build of
ow-custommatch-bot. It is based on:

- `GOOS=windows GOARCH=amd64 go list -deps ./cmd/ow-custommatch-bot`
- explicit module requirements in `go.mod`

This notice file is kept in the repository root for reference and is not
included as a GitHub Release asset.

Format
------

- Module: module path
- Version: module version used by this repository
- License: detected license name
- Copyright: copyright notice from the referenced license file
- License file: path within the cached module used as the reference source
