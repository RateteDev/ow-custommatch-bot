ow-custommatch-bot 使い方

Discord 上でカスタムマッチの募集・ランク登録・チーム振り分けを行うための利用ガイドです。初回セットアップから募集の流れ、よくあるエラーへの対処まとめています。

1. セットアップ（利用者向け）

  1. Discord Developer Portal でアプリケーションを作成し、Bot トークンを取得します。
  2. GitHub Release から ow-custommatch-bot.exe をダウンロードします。配布物は exe 1 ファイルのみです。
  3. ow-custommatch-bot.exe を起動し、画面の案内に従って BOT_TOKEN を入力します。
  4. 入力したトークンは Windows Credential Manager の ow-custommatch-bot/BOT_TOKEN に保存されます。プレイヤー情報やログは %LOCALAPPDATA%\ow-custommatch-bot\ 配下に保存されます。
  5. Discord サーバーに Bot を招待します。招待時のスコープは bot と applications.commands を選択してください。

2. コマンド一覧

コマンド 説明
/match マッチング募集を開始します。募集の開始操作は発案者のみが行います。
/register_rank チーム分け用ランクを登録または更新します。
/my_rank 登録済みランクと有効期限を確認します。

3. /match の操作フロー

  1. 発案者が /match を実行すると、募集パネルが投稿されます。
  2. 同じチャンネルにすでに募集が開いている場合は、現在の募集を閉じて新しく始めるか、そのままキャンセルするかを選べます。
  3. 参加者が「エントリー」を押します。
    • ランク未登録の場合はランク登録フローが始まり、登録完了後に自動でエントリーされます。
    • ランク登録から 30 日以上経過している場合は、再登録フローが始まります。
  4. 「取り消し」を押すとエントリーを取り消せます。
  5. 10 人以上集まったら、発案者が「振り分け」を押します。チーム分け結果と VC 招待リンクが投稿されます。
  6. 発案者は「中止」を押して募集を終了できます。

4. ランク登録について

エントリー時または /register_rank コマンドで登録できます。
ランクとディビジョン（1〜5）を選択します。Top 500 はディビジョン選択が不要です。
登録は 30 日間有効です。期限切れの場合は次回エントリー時に再登録を求められます。

5. チーム分けの仕組み

5 の倍数人数でチームを編成します。端数は「余りメンバー」として別表示されます。
ランクスコアを基準に、できるだけバランスが取れるよう振り分けます。
「再振り分け」を押すことで、何度でも再抽選できます。

6. 注意事項

  • 1 チャンネルにつき同時に開ける募集は 1 つのみです。
  • 「振り分け」「中止」を実行できるのは発案者のみです。

7. トークンの上書き

  • 起動メニューで [3] トークンを上書きする を選びます。
  • 新しい BOT_TOKEN を入力すると、保存済みトークンが更新されます。
  • 認証失敗時も、その場で新しい BOT_TOKEN を入力してそのまま再試行できます。
  • トークンの値を画面に表示する機能はありません。

8. 終了方法

  • ウィンドウを閉じるか、コンソールで Ctrl+C を押してください。

9. トラブルシュート

BOT_TOKEN が保存されていません
初回起動時のトークン入力が完了していない可能性があります。ow-custommatch-bot.exe を起動し、案内に従ってトークンを入力してください。
別のトークンに変更したい
起動メニューで [3] トークンを上書きする を選び、新しいトークンを入力してください。
Authentication failed と表示された
Discord Developer Portal を開き、BOT TOKEN を再発行または確認してください。画面の案内に従って新しいトークンを入力すると、そのままその場で再試行できます。
BOT_TOKEN の設定に失敗しました
Windows Credential Manager に保存できなかった可能性があります。詳しい手順は 使い方ページ をご確認ください。

10. 補足トラブルシュート

failed to initialize bot
%LOCALAPPDATA%\ow-custommatch-bot\ フォルダへの書き込み権限があるか確認してください。
スラッシュコマンドが表示されない
Bot 招待時に applications.commands スコープを付けたか確認してください。コマンドの反映には数分かかる場合があります。
